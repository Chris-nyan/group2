# USE CASE: <17> <Report Population Information>

## CHARACTERISTIC INFORMATION

### Goal in Context

As a data analyst, I can generate a list of all capital cities in the world ordered by population size, enabling analyses of the significance of capital cities on a global scale.
### Scope

Organization
### Level

Primary

### Preconditions

We need to have database and must know about SQL. 

### Success End Condition

Getting good output and a good report is produced to organization. 

### Failed End Condition

Output cannot be gotten because of database issues so that report cannot be produced.

### Primary Actor

Data Analyst

### Trigger

Working for the organization that requires population information.

## MAIN SUCCESS SCENARIO

1.Opening and testing world database in Xampp
2. Writing query
3. Coding with Java Language
4. Get Output for all the capital cities in the world by largest population to smallest
5. Data analyst givies report to organization
